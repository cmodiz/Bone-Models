Parameters
===============================

This module defines the parameters used in the bone mineralisation models. Each model has a specific set of parameters
governing its behavior, interaction dynamics, and biological processes.
For the bone mineralisation models, the parameters are named consistently across models to facilitate comparison and understanding.
In each parameter file, the parameter names are matched with the corresponding model nomenclature and units.

.. _ruffoni_parameters:
